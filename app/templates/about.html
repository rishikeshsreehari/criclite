About CricLite

CricLite is a live cricket score website built out of frustration with the slow loading times and complicated structure of modern cricket websites. In a world where cricket connects 2.5 billion fans, why should checking scores be such a heavy experience?

By design, CricLite is minimal and uses only plain ASCII for displaying information. The entire site is approximately 10KB with no client-side JavaScript, loading almost instantly in any environment.

Terminal Access

Get live scores directly in your terminal:

Linux/Mac: curl -s https://criclite.com/plain.txt
Windows: curl.exe -s https://criclite.com/plain.txt

Inspiration

CricLite was inspired by the design of plaintextsports.com which unfortunately didn't cover cricket and wasn't open-source. Given the massive global cricket audience, creating a similar experience for cricket fans seemed like an interesting experiment.

Technical Details

The site is built with FastAPI and hosted on an EC2 instance in AWS Mumbai servers. Content is cached via CloudFront CDN with a TTL of 1 minute to ensure the EC2 instance doesn't face excessive load during traffic spikes. All data is sourced from CricAPI with scores updating every minute.

No client-side JavaScript is used, including for the light/dark theme toggle. I do not collect any user data except some analytics via GoatCounter, which is GDPR-friendly. The theme state is stored in your browser via cookies.

Contributing

CricLite is open source software. You can view the code and contribute on GitHub.

Features in the pipeline include:

  • Detailed scorecards
  • Live commentary
  • Match analysis
  • Weather forecasts
  • Win predictors
